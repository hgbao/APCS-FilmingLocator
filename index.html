Definition

What is this?

A website using treemap and linechart to visualize filming locations for 1000 movies popularly voted following The Movie Database website. Following steps:

Take the first 1000 movies from popular movie list from The Movie Database API.

Get the IMDB code for each movie.

Go to the IMDB detail page of the movie, crawl all filming location data from that page.

The Movie Database IMDB

Environment

Language

Front-end development uses HTML, CSS, Javascript and some templates, as well as some open-source libraries. PHP, mySQL for back-end development.

Background

Real-time visualized?

The data has been taken by the author by using some functions hidden in the code, then put to the database. Therefore, the data visualized is taken from database, not real-time.

Database back-up file to import into mySQL host is included in the project.

Login

Top right corner.

Using for admin accounts to login and crawl data. Locked for security, to avoid incidentally crawling data (which usually took for an hour to full fill all data need).
